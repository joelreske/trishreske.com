# Claude Code Guidelines for StoryLines Blog

Memorial archive of Trish Reske's writing (2008-2015).

## Core Principles
- Keep it simple - minimal dependencies
- Preserve original design faithfully
- Content is the priority

## File Locations
- Posts: `src/content/posts/`
- Images: `public/images/uploads/` (by year/month)
- Theme assets: `public/images/theme/`

## Commands
- `npm run dev` - Dev server on port 4322
- `npm run build` - Build to dist/
- `npm run preview` - Preview on port 4323

## Categories
